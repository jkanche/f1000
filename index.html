Example using Epiviz Components in HTML page.

Epiviz components can be inserted in any HTML page using tags defined by the component library (e.g., epiviz-json-scatter-plot in this example). Data is supplied to the chart via the json-data attribute of the HTML tag. In this example, we show a sample JSON object representing genomic data. In this figure, we are only showing the first 5 data points although the plot renders more visual objects. When used in conjunction with epiviz-data-source components, data can be queried from a web server or via a WebSocket connection through a corresponding assignment of the json-data attribute.

Adding the epiviz element on the left to the HTML page renders the interactive scatter plot on the right.
